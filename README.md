## Getting Started

1. Clone project

    ```
    git clone https://github.com/retry19/connectin-chat-app.git
    cd connectin-chat-app
    ````

2. Copy environment file, then fill it based on your config

    ```
    cp .env.example .env
    ```
    
3. Install dependencies

    ```
    yarn install
    ```
    
    or
    
    ```
    yarn
    ```
    
4. Run project :rocket:

    ```
    yarn start
    ```
